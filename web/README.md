# chatgpt-vue3-light-mvp
## 前置条件

* Vue 3.x
* Node >= 18.12.x
* Pnpm 9.x



## 安装和运行

* 安装依赖

```bash
pnpm i
```

* 本地开发

```bash
pnpm dev
```

通过访问 `http://localhost:2048` 来查看应用。
